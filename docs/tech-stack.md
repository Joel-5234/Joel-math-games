# Tech Stack

## Core Technologies
- **HTML5**: Structure and semantic markup
- **CSS3**: Styling and responsive design
- **Vanilla JavaScript (ES6+)**: Application logic and interactivity

## Design Reference
- Reference design from `./ref/html5up-ethereal/*` for look and feel

## Testing
- **Playwright**: End-to-end UI testing
- No build tools or bundlers required

## Data Persistence
- **localStorage**: Browser-based local storage for achievements and progress

## Development Tools
- Simple HTTP server for local development (Python's http.server or similar)
- No external dependencies for the application itself

## Browser Support
- Chrome (current versions)
- Edge (current versions)
- Safari (current versions)
- Firefox (current versions)
