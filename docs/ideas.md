# Project Idea

## Parallel & Perpendicular Lines Web Trainer

A browser-based educational game built with HTML, CSS, and JavaScript to help students practice four types of line problems:
1. Slope and line description through two points
2. Relationship between two equations
3. Parallel line through a point
4. Perpendicular line through a point

The app features gamification with timers, streaks, achievements, and local progress tracking.

See `prd.md` for detailed requirements.
